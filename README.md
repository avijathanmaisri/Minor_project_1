## 📖 Overview

**GroupDNA** is a Python-based analytics tool that processes exported WhatsApp chat files and generates a comprehensive report on group communication patterns.

The application parses over **3,000 messages** from a 60-day WhatsApp group conversation, handling real-world chat complexities such as system notifications, deleted messages, media placeholders, and multi-line messages. It then transforms the raw data into meaningful statistics and behavioral insights.

This project was developed as part of the **Unlock Academy Minor Project Program**, focusing on practical applications of Python fundamentals and data analysis.

---

## ✨ Features

- 📂 Parses exported WhatsApp chat files
- 👥 Participant-wise message statistics
- 📊 Group activity overview
- 📅 Busiest day and busiest hour analysis
- 🔥 24-hour activity heatmap using NumPy
- 📝 Most frequently used words
- ⏱️ Average response time analysis
- 🤫 Longest silent streak detection
- 🎭 Personality archetype assignment
- 📄 Clean and well-structured report generation
